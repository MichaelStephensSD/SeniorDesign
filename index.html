MAS User Guide and Manual

Login and Sign Up

The web application allows users to create an account or log in to an existing account. To access these options, follow these steps:

  1. Click on the "Login" or "Sign Up" button located on the top right corner of the page
  2. If you choose "Sign Up," fill in the required information such as your email and desired password
  3. Click the "Create Account" button to create your account
  4. If you choose "Login," enter your registered email and password
  5. Click the "Login" button to access your account

Note: Having an account allows users to save their data and settings for future use. It is recommended to use a strong and unique password for security purposes.

Displaying Data with a Ticker Symbol

The web application allows users to display data for a specific stock by entering its ticker symbol. To display data for a stock, follow these steps:

  1. Enter the ticker symbol for the desired stock in the top left bar
  2. Click on the "Display" button
  3. The chart for the stock will be displayed on the page

Note: The ticker symbol must be entered correctly and match the symbols available in the database of the web application. If the symbol is incorrect, no data will be displayed.

Switching Timeframes

The web application allows users to view data in different timeframes. To switch the timeframe, follow these steps:

  1. Click on the "Timeframe" dropdown located on the top bar of the page
  2. Select the desired timeframe from the list of options
  3. The chart will be updated to display data within the selected timeframe

Note: The available timeframes may vary depending on the data being viewed. Some data may only be available in certain timeframes.

Drawing Lines and Shapes

The web application allows users to draw shapes on the chart for analysis purposes. To draw a shape, follow these steps:

  1. Click on the "Draw" button located on the left hand toolbar
  2. Select the type of shape you would like to draw (e.g. line, rectangle, etc.)
  3. Click and drag on the chart to draw the shape

Note: You can adjust the properties of the shape (e.g. color, width) by right-clicking on the shape and selecting "Properties".

Selecting an Indicator

The web application allows users to add indicators to the chart for analysis purposes. To add an indicator, follow these steps:

  1. Click on the "Indicators" dropdown located at the top of the page
  2. Select the indicator you would like to add (e.g. Moving Average, Bollinger Bands, etc.)
  3. Configure the properties of the indicator as desired
  4. Click the "Apply" button to add the indicator to the chart

Note: The web application provides a variety of technical indicators that can be used to analyze the data on the chart. Some of the indicators include moving averages, Bollinger Bands, RSI, MACD, and many others. Users can experiment with different indicators to find the one that works best for their analysis. However, it is important to note that using too many indicators can lead to confusion and may not provide a clear picture of the data. It is recommended to use only a few indicators and use them in conjunction with other analysis tools such as trend lines and chart patterns.

Setting Up Price Trigger Alerts

The web application allows users to receive notifications when a stock reaches a certain price. To set up a price trigger alert, follow these steps:

  1. Click on the "Alerts" button located on the top right corner of the page
  2. Type in the ticker symbol of the stock you would like to receive alerts for
  3. Enter the price you would like to receive an alert for
  4. Click the "Set Alert" button to set the price trigger alert

Note: You can view and manage your alerts by clicking on the "Alerts" button and then selecting "View Alerts".

User Watchlist

The web application allows users to keep track of stocks that they are interested in. To add a stock to your watchlist, follow these steps:

  1. Click on the "Watchlist" button located on the top right corner of the page
  2. Type in the ticker symbol of the stock you would like to add to your watchlist
  3. Click the "Add" button to add the stock to your watchlist

Note: You can view your watchlist by clicking on the "Watchlist" button and then selecting "View Watchlist".
